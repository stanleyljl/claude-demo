# 游戏程序员简历模板

这是一个现代化的游戏程序员简历HTML模板，专为游戏开发工程师设计，展示游戏开发技能和响应式设计。

## 项目特点

- 🎨 **现代设计** - 使用CSS变量和渐变背景
- 📱 **响应式布局** - 适配桌面、平板和手机
- 🚀 **性能优化** - 轻量级，快速加载
- 🖨️ **打印友好** - 支持直接打印
- 🌙 **深色主题** - 优雅的配色方案
- 🎮 **游戏开发优化** - 突出游戏开发技能和项目展示

## 文件结构

```
claude-demo/
```

## 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/stanleyljl/claude-demo.git
   ```

2. 在浏览器中打开 `index.html` 文件查看简历

3. 如需打印，点击页面右下角的打印按钮

## 自定义

要自定义此游戏程序员简历，请编辑 `index.html` 文件：

1. **个人信息** - 更新姓名、职位、联系方式
2. **工作经历** - 修改游戏公司、职位、时间描述
3. **技能专长** - 添加或删除游戏开发相关技能标签（游戏引擎、编程语言、图形API等）
4. **项目经验** - 更新游戏开发项目、独立游戏作品
5. **教育背景** - 更新学校、专业、时间
6. **证书荣誉** - 更新游戏开发相关的证书和奖项
7. **样式调整** - 修改CSS变量来改变颜色主题

### 主要CSS变量

```css
:root {
    --primary-color: #3498db;      /* 主色调 */
    --secondary-color: #2c3e50;    /* 次要色调 */
    --accent-color: #e74c3c;       /* 强调色 */
    --light-color: #f8f9fa;        /* 浅色 */
    --dark-color: #343a40;         /* 深色 */
}
```
